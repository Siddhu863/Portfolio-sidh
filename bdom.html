BOM & DOM Practice

BOM (Browser Object Model) and DOM (Document Object Model) are two core concepts in JavaScript for interacting with the browser and web pages.

BOM allows you to interact with browser features such as window, location, history, navigator, and more.
DOM allows you to interact with and manipulate the structure, style, and content of HTML documents dynamically using JavaScript.

Key Differences:
BOM DOM
Represents browser components outside the page content (window, history, location, navigator, screen, etc.) Represents the structure of the HTML document (elements, attributes, text, etc.)
Used for browser-level operations (open new tab, get URL, alert, etc.) Used for page-level operations (change content, add/remove elements, style changes, etc.)
Not standardized by W3C Standardized by W3C
Examples: window.open(), window.location, alert() Examples: document.getElementById(), element.innerHTML, createElement()
